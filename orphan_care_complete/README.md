# 🏠 مشروع كَنَفْ - منصة متكاملة لرعاية الأيتام

## 📋 نظرة عامة

**كَنَفْ** هي منصة رقمية متكاملة تجمع بين تطبيق موبايل احترافي ومنظومة إدارية قوية، مصممة لتسهيل إدارة دور رعاية الأيتام وتنسيق التبرعات والتطوع.

### 🎯 الأهداف الرئيسية

*   **تسهيل التبرعات:** توفير منصة آمنة وسهلة للمتبرعين لتقديم مساعداتهم المالية والعينية.
*   **تنظيم التطوع:** إدارة فرص التطوع وتتبع ساعات العمل والشهادات.
*   **إدارة الأيتام:** توثيق شامل لحالات الأيتام والاحتياجات والكفالات.
*   **التقارير والإحصائيات:** توليد تقارير مفصلة عن الأداء والتبرعات.

## 🚀 البدء السريع

### المتطلبات

**للتطبيق (Frontend):**
*   Flutter SDK 3.0+
*   Dart 3.0+

**للمنظومة (Backend):**
*   Python 3.10+
*   pip (مدير المكتبات)

### التثبيت

#### 1️⃣ تثبيت المنظومة (Backend)

```bash
cd backend/orphan_care_backend

# إنشاء بيئة افتراضية
python -m venv venv

# تفعيل البيئة الافتراضية
# على Windows:
venv\Scripts\activate
# على macOS/Linux:
source venv/bin/activate

# تثبيت المكتبات المطلوبة
pip install -r requirements.txt

# تطبيق الهجرات
python manage.py migrate

# إنشاء مستخدم إداري
python manage.py createsuperuser

# تشغيل الخادم
python manage.py runserver
```

#### 2️⃣ تثبيت التطبيق (Frontend)

```bash
cd frontend/orphan_care_app

# تثبيت المكتبات
flutter pub get

# تشغيل التطبيق على محاكي
flutter run

# أو بناء تطبيق الويب
flutter run -d chrome
```

## 📱 الميزات الرئيسية

### للمتبرعين (Donors)
*   ✅ استعراض احتياجات الأيتام
*   ✅ تقديم تبرعات مالية وعينية
*   ✅ تتبع سجل التبرعات
*   ✅ الحصول على إشعارات بحالة التبرعات

### للمتطوعين (Volunteers)
*   ✅ استعراض فرص التطوع المتاحة
*   ✅ التقديم والتسجيل في الفرص
*   ✅ تتبع ساعات العمل والنقاط
*   ✅ الحصول على شهادات التطوع

### لدور الرعاية (Care Homes)
*   ✅ لوحة تحكم شاملة
*   ✅ إضافة وإدارة احتياجات الأيتام
*   ✅ إدارة المتطوعين والزيارات
*   ✅ تتبع التبرعات الواردة
*   ✅ إدارة المخزن والموارد
*   ✅ توليد التقارير والإحصائيات

## 🔌 واجهات برمجية (APIs)

### المصادقة

```http
POST /api/auth/login/
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "password123"
}
```

### الأيتام

```http
GET /api/orphans/                    # الحصول على قائمة الأيتام
GET /api/orphans/{id}/               # الحصول على تفاصيل يتيم
POST /api/orphans/                   # إضافة يتيم جديد
PUT /api/orphans/{id}/               # تحديث بيانات يتيم
DELETE /api/orphans/{id}/            # حذف يتيم
GET /api/orphans/statistics/         # إحصائيات الأيتام
```

### التبرعات

```http
GET /api/donations/                  # الحصول على قائمة التبرعات
POST /api/donations/                 # إنشاء تبرع جديد
GET /api/donations/my-donations/     # التبرعات الخاصة بي
GET /api/donations/statistics/       # إحصائيات التبرعات
```

### المتطوعين

```http
GET /api/volunteers/                 # الحصول على قائمة المتطوعين
POST /api/volunteers/apply/          # التقديم للتطوع
GET /api/volunteers/statistics/      # إحصائيات المتطوعين
```

### الإحصائيات والتقارير

```http
GET /api/stats/dashboard/            # إحصائيات لوحة التحكم
GET /api/reports/                    # التقارير الشاملة
```

## 🎨 التصميم والواجهات

### نمط التصميم: Glassmorphism
التطبيق يستخدم نمط التصميم الحديث **Glassmorphism** الذي يجمع بين:
*   تأثيرات زجاجية شفافة
*   ألوان متدرجة ناعمة
*   حركات انتقالية سلسة
*   واجهة مستخدم بديهية

### الألوان الأساسية
*   🟠 البرتقالي الرئيسي: `#FF9500`
*   ⚫ الأسود الداكن: `#131313`
*   ⚪ الأبيض مع شفافية
*   🟢 الأخضر الزمردي: `#10B981`

### الخطوط المستخدمة
*   **Cairo:** للنصوص الرئيسية
*   **Tajawal:** للنصوص الثانوية
